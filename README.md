# вҡҷпёҸ FullStack Project вҖ“ How to Run the Project

## рҹ–ҘпёҸ Frontend (React)

### 1пёҸвғЈ rm -rf node_modules
### 2пёҸвғЈ rm package-lock.json
### 3пёҸвғЈ npm cache clean --force 
### 4пёҸвғЈ npm install
### 5пёҸвғЈ npm install react-slick slick-carousel
### 6пёҸвғЈ npm install react-leaflet leaflet
### 7пёҸвғЈ npm install react-router-dom
### 8пёҸвғЈ npm run dev



## рҹӣ пёҸ Backend (Laravel)

### 1пёҸвғЈ composer install
### 2пёҸвғЈ php artisan migrate
### 3пёҸвғЈ php artisan serve
